# 桌面清洁赛项

## 业务流程

1. 建图，保存地图，示教记录方便抓取的点；
2. movebase导航到第一个桌子前；
3. 颜色识别提取待抓取物块颜色重心，机器人轻微平移调整，对准物块；
4. 机械爪张开，机械臂规划动作，抓取物块，机械臂归位；
5. 根据里程计信息，平移一段距离，到垃圾桶前方，机械臂规划动作，放置物块，机械臂归位，自旋180度；
6. movebase导航到第二个桌子前；
7. 颜色识别提取待抓取物块颜色重心，机器人轻微平移调整，对准物块；
8. 机械爪张开，机械臂规划动作，抓取物块，机械臂归位；
9. 根据里程计信息，平移一段距离，到垃圾桶前方，机械臂规划动作，放置物块，机械臂归位，自旋180度；
10. movebase导航返回出发点；


## 示例代码描述

- tf_listener：获取当前坐标的坐标点；
- movebase_client：导航到第一张桌子，第二张桌子，返回出发点；
- color_detect：颜色识别提取重心，机器人对齐颜色重心；
- arm_grab：机械臂运动，抓取物块，机械臂归位；
- arm_put：机器人平移，导航到垃圾桶前方，机械臂规划，打开手爪，放置物块，机械臂归位，机器人自旋180度；
